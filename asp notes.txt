Page lifecycle of ASP.NET:

S - Start
I - Initialized
L - Load
V - Validate
E - Event Handling
R - Render

State Management Techniques

Http is a stateless protocol, we need to maintain the state of a control

Client side
1. ViewState
2. Hiddenfield
3. Cookies
4. Querystring

Server side
1. Session
2. Application
